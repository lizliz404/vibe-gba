vibe-gba

A scratch-built Game Boy Advance emulator running in your browser. Load a .gba or zipped ROM and play with keyboard controls. No ROM is included — this is an emulator, not a ROM distributor.

Project page

Select a .gba ROM to begin.

Keyboard: Arrows / WASD = D-pad, Z = A, X = B, Enter = Start, Backspace = Select, Q / E = L / R, Esc = pause.
